# Moon Grove Quest

A small original top-down fantasy adventure game for the browser, built as a static site suitable for GitHub Pages.

## Play
- Arrow keys / WASD: move
- Space / J: sword slash
- Enter: start / restart
- Mobile: use the on-screen D-pad and attack button

## Goal
Collect the golden key, unlock the north gate, defeat or dodge monsters, and reach the glowing shrine.

## Notes
This is inspired by classic top-down adventure games in broad genre only. Characters, setting, names, and generated assets are original.
